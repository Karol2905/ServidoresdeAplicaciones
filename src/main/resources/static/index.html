MicroSpringBoot By Karol

Servidor web mínimo construido con Java Reflection e IoC.

Endpoints disponibles

  • / — Saludo raíz
  • /hello — Hello World
  • /pi — Valor de PI
  • /greeting — Saludo (World por defecto)
  • /greeting?name=Carlos — Saludo personalizado

Archivos estáticos

  • /index.html — Esta página HTML
  • /logo.png — Imagen PNG

Sin dependencias externas — solo Java 21.
